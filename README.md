#  Portafolio Personal — Isaac David Ortiz Guzmán

Portafolio web personal desarrollado con HTML, CSS y JavaScript vanilla. Presenta información sobre mí, mis proyectos y una sección de contacto.

---

##  Estructura del proyecto

portfolio/
├── index.html       # Página principal del portafolio
├── pets.html        # Galería de mascotas
├── styles.css       # Estilos globales
├── script.js        # Lógica e interactividad
└── img/             # Imágenes locales

---

##  Funcionalidades

- **Sobre mí** — Sección con texto expandible/colapsable mediante un botón.
- **Proyectos** — Tarjetas con descripción y enlace a GitHub de cada proyecto, con opción de ocultar/mostrar la lista.
- **Contacto** — Formulario básico con campos de nombre y correo.
- **Mascotas** — Página secundaria con galería de imágenes.
- **Diseño responsive** — Adaptado para móvil, tablet y escritorio con media queries.

---

##  Responsive Design

El sitio está adaptado para tres rangos de pantalla:

- **Móvil** (`< 480px`) — Navegación en columna, tarjetas en una sola columna.
- **Tablet** (`481px – 768px`) — Tarjetas en dos columnas.
- **Escritorio** (`> 769px`) — Tarjetas en tres columnas.
